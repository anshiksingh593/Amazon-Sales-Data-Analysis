# Amazon-Sales-Data-Analysis
Amazon Sales Data Analysis project using Python and Excel. Performed data cleaning, preprocessing, feature engineering, and built interactive dashboards to generate business insights from e-commerce sales data.


# Amazon Sales Data Analysis

## Project Overview
This project analyzes Amazon sales data using Python and Excel. The objective is to transform raw sales data into meaningful business insights through data cleaning, preprocessing, and visualization.

## Project Workflow

1. Data Collection
   - Dataset downloaded from Kaggle

2. Data Cleaning & Processing
   - Performed using Python in Jupyter Notebook
   - Removed duplicates
   - Handled missing values
   - Converted date formats
   - Created new features

3. Data Visualization
   - Created dashboards in Microsoft Excel
   - Used Pivot Tables, Charts, and KPIs
